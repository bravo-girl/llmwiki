## Seite 2

Ab dem 01.01.2026 müssen alle Bahnakteure innerhalb der EU
auf den Organisation Code umstellen
Wechsel in Zuständigkeit
• Bis 31.12.2025 gilt für alle Bahnakteure innerhalb der EU der 4-stellige numerische Company Code, vergeben von der UIC,
zur eindeutigen Identifizierung im TAF/TAP-Nachrichtenaustausch
• Ab dem 01.01.2026 müssen alle Bahnakteure auf einen 4-stelligen alphanumerischen Organisation Code, vergeben
von der ERA, umstellen
Keine automatische Migration!
Damit Sie ab dem 01.01.2026 auch weiterhin problemlos am TAF/TAP-Nachrichtenaustausch mit
! !
Ihren Partnern teilnehmen können, benötigen Sie einen Organisation Code, der gleichlautend zu
Ihrem bisherigen Company Code ist. Wenn Sie bisher keinen Code haben, beantragen Sie bitte
umgehend einen Organisation Code und melden ihn an Ihre:n Kundenberater:in bzw. Ihre Partner im
TAF/TAP-Meldungsaustausch.

## Seite 3

Bei der Umstellung auf den Organisation Code gibt es folgenden
Handlungsbedarf zu beachten
Company Code vorhanden aber kein Organisation Code 
Aktion: Bitte gleichlautenden Organisation Code beantragen
Company Code und Organisation Code sind vorhanden und weichen voneinander ab 
Aktion: Organisation Code löschen lassen und danach einen gleichlautenden Organisation Code beantragen
Company Code und Organisation Code sind gleichlautend 
Aktion: Keine Aktion erforderlich
Weder Company Code noch Organisation Code vorhanden 
Aktion: Organisation Code beantragen
Bitte melden Sie zur Nachvollziehbarkeit die Angleichung des Organisation Codes umgehend an Ihre:n
Kundenberater:in und weiteren Partnern im TAF/TAP-Meldungsaustausch !!
Ebenso bei Neuvergabe und Erstvergabe !!
